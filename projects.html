Vanilla Weather App

Built using HTML, advanced CSS, JavaScript, and an active API

This weather app was a project developed during an 8-week advanced JavaScript workshop, “SheCodes Plus.” Building on the foundation of knowledge gained from the “SheCodes Basics” workshop, I became skilled in development workflow using GitHub and GitHub Desktop, hosting on Netlify, Bootstrap, APIs and advanced JavaScript techniques. Using all these skills, I built a Vanilla JavaScript Weather App that uses an API from OpenWeatherMap to extract weather data. In addition, it provides the current weather, high/lows for the day, windspeed, humidity as well as an extended 5-day weather forecast for the selected or searched city.
